# How to make app with vue.js

- npm i -g @vue/cli
- vue --version
- vue create \<appName>
- cd appName
- npm run serve
- or vue ui
- Extension: Vetur
- Devtools - Vue - Can see "Components Structure"

# Basic Structure

1. \<template>\</template>
2. \<script>import Component from './'; export default{name: 'app', components:{}, data(){}};\</script>
3. \<style>\</style> Or \<style scoped>\</style>
